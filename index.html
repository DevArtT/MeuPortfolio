Olá! Sou Arthur Telles, um entusiasta do desenvolvimento web de 21 anos. Atualmente, estou aprofundando meus conhecimentos em HTML e CSS através do curso "HTML e CSS: ambientes de desenvolvimento, estrutura de arquivos e tags" da Alura. Estou dedicado a transformar minhas ideias em interfaces intuitivas e funcionais.

Conhecimentos em Destaque:

  • Estrutura fundamental do HTML (< !DOCTYPE html>, head, body)
  • Uso de tags essenciais para conteúdo (h1, p, img, ul, ol, li)
  • Configuração de meta tags (charset) e títulos de página
  • Organização semântica do conteúdo com diferentes níveis de cabeçalho
  • Introdução à acessibilidade com o atributo "alt" em imagens
  • Noções básicas de como o navegador renderiza o HTML (impacto do DOCTYPE)
